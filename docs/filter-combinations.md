# Filter Combinations Reference

This document lists all possible data retrieval combinations for the job market dashboard.

## Filter Dimensions

| Dimension | Options | Count |
|-----------|---------|-------|
| **Category** | Java, Data | 2 |
| **Metric Type** | Total, With Salary, Remote, Remote + Salary | 4 |
| **City** | All, Wrocław, Śląsk | 3 |
| **Experience Level** | All Levels, Junior, Mid, Senior | 4 |
| **Salary Range** | Any Salary, < 25k, 25-30k, > 30k | 4 |

## Total Combinations

- **Per Category**: 4 × 3 × 4 × 4 = **192 combinations**
- **All Categories**: 192 × 2 = **384 combinations**

## API Query Parameters Mapping

| Filter | API Parameter | Values |
|--------|---------------|--------|
| Metric Type | `metricType` | `TOTAL`, `WITH_SALARY`, `REMOTE`, `REMOTE_WITH_SALARY` |
| City | `city` | `null` (All), `wroclaw`, `slask` |
| Experience | `experienceLevel` | `null` (All), `JUNIOR`, `MID`, `SENIOR` |
| Salary | `salaryRange` | `null` (Any), `UNDER_25K`, `RANGE_25_30K`, `OVER_30K` |

## Example API Calls

```bash
# Java / Total / All / All Levels / Any Salary
GET /api/stats/java?metricType=TOTAL

# Java / With Salary / Wrocław / Mid / 25-30k
GET /api/stats/java?metricType=WITH_SALARY&city=wroclaw&experienceLevel=MID&salaryRange=RANGE_25_30K

# Java / Remote + Salary / Śląsk / Senior / > 30k
GET /api/stats/java?metricType=REMOTE_WITH_SALARY&city=slask&experienceLevel=SENIOR&salaryRange=OVER_30K
```
